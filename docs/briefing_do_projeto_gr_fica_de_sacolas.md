# Projeto: Landing Page Gráfica de Sacolas Personalizadas
## Objetivo
Captação de leads qualificados através de um configurador de produto "Mobile-First" para uma gráfica especializada em sacolas premium.

## Estrutura da Página
1. **Hero Section:** Título de alta conversão, subtítulo técnico (resistência 4kg) e CTA âncora.
2. **Configurador (Passo a Passo):**
   - Tamanhos (P, M, G, GG)
   - Papéis (Offset, Duplex, Kraft)
   - Acabamentos (Laminação Fosca/Brilho)
   - Impressão (1x0, 4x0 + Pantone)
   - Alças (Papel/Tecido)
   - Ilhós (Branco/Preto)
3. **Resumo Sticky:** Barra inferior com resumo das escolhas e botão "Solicitar Orçamento".

## Diretrizes Visuais
- Dark Mode / CMYK Accents.
- Feedback visual imediato ao selecionar opções.
- UX de e-commerce gráfico profissional.
